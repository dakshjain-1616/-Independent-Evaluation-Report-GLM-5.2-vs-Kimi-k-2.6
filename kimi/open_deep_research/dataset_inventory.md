# Dataset and Artifact Inventory

## Summary

This document inventories all datasets, result artifacts, prompt collections, configuration files, and example outputs within the `open_deep_research` repository. Each entry includes purpose, source, schema, dependencies, and identified quality concerns.

| Attribute | Detail |
|-----------|--------|
| **Name** | Deep Research Bench |
| **Location** | LangSmith (public link referenced in README) |

| **Quality Concerns** | 1. Schema not versioned locally â€” drift risk if LangSmith dataset is modified.<br>2. No local backup or snapshot â€” evaluation reproducibility depends on LangSmith availability.<br>3. 22-field schema is opaque in repository; no documentation mapping fields to evaluation dimensions.<br>4. Chinese prompts (50%) may have encoding or judge-model bias not accounted for in RACE scoring. |
